What is Prompt2PicBot?

Prompt2PicBot is a Telegram bot that lets users generate images with text prompts using AI models like FLUX.1-dev on Hugging Face. Built with love, Tailwind, Cloudflare, Supabase, and a little bit of magic ✨

Due to free-tier limits on Hugging Face, the live image generation is currently paused. But the backend, code, and demo are fully ready for review.
